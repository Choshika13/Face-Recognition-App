**Real-Time Facial Verification System**

A deep learning-based facial verification application that uses a Siamese Neural Network to compare real-time webcam input against a set of authorized anchor images.

**Features:**

Siamese Network Architecture: Leverages contrastive loss and distance-based metric learning to distinguish between faces.

Custom Data Pipeline: Built using tf.data for efficient image preprocessing, normalization, and batching.

Real-Time Verification: Integrated with OpenCV and Kivy to provide a seamless, live-feed GUI for identity authentication.

Configurable Sensitivity: Includes adjustable detection and verification thresholds to balance security and usability.

**Technologies Used:**

Languages: Python

Deep Learning: TensorFlow, Keras

Computer Vision: OpenCV

GUI: Kivy

Environment: Jupyter Notebook
